# Basireddy Ganesh Portfolio

Static portfolio website prepared for GitHub Pages.

## Deploy to GitHub Pages

1. Create a public GitHub repository named:

```text
ganeshbasireddy2505.github.io
```

2. Upload these files and folders to the repository root:

```text
index.html
style.css
script.js
assets/
```

3. Commit the files.

4. Open:

```text
https://ganeshbasireddy2505.github.io
```

If it shows 404, wait a few minutes and refresh. If needed, go to repository Settings > Pages and select `main` branch and `/root`.
